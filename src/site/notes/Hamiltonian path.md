# [[Hamiltonian path\|Hamiltonian path]]

```ad-Definizione
title: Hamiltonian path

A **Hamiltonian path** is any [[00 - Graph - MM#Path|path]] in a [[00 - Graph - MM|graph]] that visits each and every node exactly once.
```
