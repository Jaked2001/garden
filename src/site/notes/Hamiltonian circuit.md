# [[Hamiltonian circuit\|Hamiltonian circuit]]

```ad-Definizione
title: Hamiltonian circuit

A **Hamiltonian circuit** or **cycle** is any [[00 - Graph - MM#Cycle|cycle]] in a [[00 - Graph - MM|graph]] that visits each and every node exactly once.

```
