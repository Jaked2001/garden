# [[Eulerian path\|Eulerian path]]

```ad-Definizione
title: Eulerian path

An Eulerian path is any [[00 - Graph - MM#Path|path]] in a [[00 - Graph - MM|graph]] that visits each and every link (or edge) exactly once.
```
