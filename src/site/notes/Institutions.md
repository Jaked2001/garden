# [[Institutions\|Institutions]]

```ad-Definizione
title: Institution

An **institution** is a set of socially embedded and accepted systems of [[#Rules]].

```

Institutions can be:
- [[#Regulative institution]]
- [[#Normative institution]]
- [[#Cognitive institution]]
## Regulative institution

```ad-Definizione
title: Regulative institution

A **regulative institution** is an [[#Institutions|institution]] that exists through laws and regulation. 

```


It's enforced through coercive forces.

## Normative institution

```ad-Definizione
title: Normative institution

A Normative institution apply a logic of appropriateness and compatibility and is sanctioned through social exclusion.

```


## Cognitive institution

```ad-Definizione
title: Cognitive institution

A **cognitive institution** is based on mimetic forces in places in environments of high uncertainty.

```
